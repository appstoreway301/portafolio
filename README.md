# Portafolio

Portafolio personal hecho con **Next.js**. Toda la información se edita desde un solo archivo.

## ✏️ Cómo personalizarlo

1. Abre `data/config.js`.
2. Cambia tu **nombre, título, resumen, correo y links** (GitHub, LinkedIn).
3. Edita la lista de **proyectos**: nombre, descripción, tecnologías, link al repo y a la demo.
4. (Opcional) Pon imágenes de tus proyectos en la carpeta `public/` y referéncialas en `imagen` (ej: `"/classbland.png"`).
5. (Opcional) Pon tu CV en PDF en `public/cv.pdf` y en config: `cv: "/cv.pdf"`.

## ▶️ Ejecutar en local

```bash
npm install
npm run dev
```

Abre http://localhost:3000

## 🚀 Publicarlo gratis en Vercel

1. Sube este proyecto a GitHub.
2. Entra a https://vercel.com y regístrate con tu cuenta de GitHub.
3. Click en **Add New → Project**, elige este repositorio y dale a **Deploy**.
4. En 1–2 minutos tendrás un link público tipo `tunombre.vercel.app`.

Ese link es el que pones en tu CV. ✅
